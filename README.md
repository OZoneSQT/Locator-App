# Locator-App
Android app to display Jid Grid from gps

The compleated version of an exam project, as I intenden, without the exam requierments.
https://github.com/OZoneSQT/jidGrid

 - Permission for Internet connection is only used to save on Google-Drive.
 - App has no log collection.
 - Permission for Location, is local on the device to retrive Location from device GPS.
 - Permission for device sensores, if sensores are pressent on the device, to store additional temperatur, humidity, and pressure.
